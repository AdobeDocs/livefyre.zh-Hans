# 将帖子添加到导航指南{#add-posts-to-the-navigation-guide}

将帖子添加到Storify 2中的导航指南。

1. 在Storify 2中写文章。
1. 为帖子编写标题。 在添加标题之前，不能显示帖子的导航高亮显示。
1. 单击 **[!UICONTROL Navigation Guide]** 帖子上的图标以将其添加到导航指南。
1. 确认要将帖子添加到导航指南。
