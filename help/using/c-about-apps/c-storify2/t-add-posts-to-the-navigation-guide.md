# 将帖子添加到导航指南{#add-posts-to-the-navigation-guide}

在Storify中的导航指南中添加帖子。

1. 在Storify中编写帖子。
1. 为帖子编写标题。添加标题之前，不能显示帖子的“导航突出显示”。
1. 单击帖子上的 **[!UICONTROL Navigation Guide]** 图标，将其添加到导航指南。
1. 确认要将该帖子添加到导航指南。
