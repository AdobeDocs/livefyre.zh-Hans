# 为导航指南添加标题{#add-a-title-to-the-navigation-guide}

为您的导航指南提供标题，以便向站点访客清晰显示内容。

1. 打开Storify故事。
1. 在Storify App Designer中，切换 **[!UICONTROL Navigation Guide]** 切换。
1. 在字段中输入标题，填充标题“添加标题”。”
