# 将标题添加到导航指南{#add-a-title-to-the-navigation-guide}

为导航指南提供一个标题，以便向网站访客明确内容。

1. 打开Storify 2文章。
1. 在Storify 2 App Designer中，打开&#x200B;**[!UICONTROL Navigation Guide]**&#x200B;切换。
1. 在填充了“添加标题”字样的字段中输入标题。
