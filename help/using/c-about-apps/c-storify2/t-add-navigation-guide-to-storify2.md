# 将导航指南添加到故事2{#add-a-navigation-guide-to-a-storify}

在Storify 2应用程序中添加导航指南，以在您的故事旁边的一个突出位置创建经过链接的帖子列表。

1. 打开Storify 2文章。
1. 在Storify 2 App Designer中，打开切换 **[!UICONTROL Navigation Guide]** 键。
