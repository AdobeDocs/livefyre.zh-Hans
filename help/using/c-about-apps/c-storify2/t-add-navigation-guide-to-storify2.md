# 将导航指南添加到Storify 2{#add-a-navigation-guide-to-a-storify}

为Storify 2应用程序添加导航指南，以在您的文章旁边的一个突出位置创建对等帖子的列表。

1. 打开Storify 2文章。
1. 在Storify 2 App Designer中，打开&#x200B;**[!UICONTROL Navigation Guide]**&#x200B;切换。
