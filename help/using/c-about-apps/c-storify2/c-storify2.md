# Storify2{#storify}

使用Storify从实时内容创建内容丰富的交互式故事。

Storify将领先的社交故事平台引入Livefyre Studio，使用户能够轻松地从Web中收集帖子、照片、视频、音频和媒体，并创建可嵌入任何网站的故事。借助Storify2，任何人都可以轻松快速地合并和排列实时内容，从而创建出交互、动态和社交的新故事格式。

使用Storify从网络中收集社交内容和丰富媒体，创建引人入胜的动态故事。
