# 移动导航指南{#move-the-navigation-guide}

指定导航指南在屏幕上的位置。

1. 在Storify App Designer中，切换 **[!UICONTROL Navigation Guide]** 切换。
1. 从下 **[!UICONTROL Orientation]** 拉菜单中选择顶部、左侧或右侧。
