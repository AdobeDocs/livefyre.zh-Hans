# 移动导航指南{#move-the-navigation-guide}

在屏幕上指定导航指南的位置。

1. 在Storify 2 App Designer中，打开&#x200B;**[!UICONTROL Navigation Guide]**&#x200B;切换。
1. 从&#x200B;**[!UICONTROL Orientation]**&#x200B;下拉列表中选择top、left或right。
