# 移动导航指南{#move-the-navigation-guide}

在屏幕上指定导航指南的位置。

1. 在Storify 2 App Designer中，打开切换 **[!UICONTROL Navigation Guide]** 键。
1. 从下拉列表中选择顶部、左 **[!UICONTROL Orientation]** 侧或右侧。
