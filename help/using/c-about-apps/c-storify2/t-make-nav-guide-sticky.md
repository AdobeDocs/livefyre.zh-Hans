# 使导航指南变得粘滞{#make-the-navigation-guide-sticky}

选择使导航指南粘贴到屏幕的特定区域。

1. 在Storify 2 App Designer中，打开切换 **[!UICONTROL Navigation Guide]** 键。
1. 打开切 **[!UICONTROL Stickiness]** 换。
