# 功能卡{#feature-card}

将特色内容添加到您的页面。

使用功能卡在一张卡上突出显示个别内容。应用程序优先考虑最新的特色内容，并循环浏览最新的项目，以确保相关的全新显示。

功能卡将显示最近的特色物品(不一定是最近发布的项目)，允许您选择最佳内容以便及时在页面上演示。
