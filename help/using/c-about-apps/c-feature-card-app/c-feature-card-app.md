# 功能卡{#feature-card}

将单个特色内容添加到您的页面。

使用功能卡在单张卡显示屏上突出显示个别内容。 应用程序优先处理最新的特色内容，并循环查看最新项目以确保显示新鲜且相关。

功能卡将显示最近发布的特色项目（不一定是最近发布的项目），允许您选择最佳内容，在页面上及时进行演示。
