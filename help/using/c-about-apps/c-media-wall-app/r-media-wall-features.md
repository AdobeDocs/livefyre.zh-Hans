# 媒体墙功能{#media-wall-features}

媒体墙应用程序为您的站点提供这些功能。



* 根据发布的日期(而非创建内容的日期)按顺序显示内容。
* 将Livefyre和社交媒体内容组合在一个显示屏中。
* 允许用户喜欢内容，并将文本或照片直接添加到媒体墙。
* 允许用户将内容共享到Twitter和Facebook。共享的共享内容将重定向那些将其返回到媒体墙上的内容的人员。
* 允许您为特色内容创建特殊样式。
