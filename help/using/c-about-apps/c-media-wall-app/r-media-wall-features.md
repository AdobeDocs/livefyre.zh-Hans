# 媒体墙功能{#media-wall-features}

媒体墙应用程序为您的站点提供这些功能。



* 根据内容发布日期（而非内容创建日期）按顺序显示内容。
* 将Livefyre和社交媒体内容组合到一个显示屏中。
* 允许用户喜欢内容，并直接将文本或照片添加到媒体墙。
* 允许用户将内容共享到Twitter和Facebook。 共享权限会将单击该权限的用户重定向回您的媒体墙上的内容。
* 允许您为特色内容创建特殊的样式。
