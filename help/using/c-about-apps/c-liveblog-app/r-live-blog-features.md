# 实时博客功能{#live-blog-features}

“实时博客应用程序”为您的站点优惠以下功能。



* 通过直接在实时博客上发布实时更新。
* 创建规则来管理社交内容，并搜索要添加到博客的特定精选内容。
* 从移动设备更新您的实时博客。
* 启用对每个博客更新的回复，鼓励围绕每个实时博客更新增加对话和参与。
* 上传照片并将自定义HTML添加到顶级帖子
