# 上传按钮{#upload-button}

上传按钮为站点访客提供了一种上传内容的方法。

您可以直接从您的站点收集用户生成的内容，并在任何Livefyre应用程序中显示该内容。 通过鼓励网站访客通过上传内容为对话做出贡献来吸引他们。 Livefyre支持图像、视频和音频内容。
