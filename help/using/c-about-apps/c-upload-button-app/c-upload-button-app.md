# 上传按钮{#upload-button}

上传按钮为站点访客提供了一种上传内容的方法。

您可以直接从您的站点收集用户生成的内容，并在任何Livefyre App中显示该内容。鼓励访客通过上传内容来促进对话，从而吸引访客。Livefyre支持图像、视频和音频内容。
