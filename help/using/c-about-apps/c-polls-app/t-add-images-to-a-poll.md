# 向投票中添加图像{#add-images-to-a-poll}

通过添加图像为投票增添视觉趣味。

1. 单击问题或答案右侧的图标。
1. 将图像拖放到提供的窗口中，或使用Filepicker从Instagram、Flickr、Dropbox或列出的其他提供商中选择图像。
1. 根据需要裁剪图像。
1. 单击 **[!UICONTROL Save]** 以将图像添加到投票中。
1. 将鼠标悬停在图像上以查看问题或选择，然后单击显示的图标以裁剪、删除或替换图像。
