# 将图像添加到投票{#add-images-to-a-poll}

添加图像，为您的投票增添视觉趣味。

1. 单击问题或答案右侧的图标。
1. 将图像拖放到提供的窗口中，或使用FilePicker从Instagram、Flickr、Dropbox或列出的其他提供商之一中选择图像。
1. 根据需要裁剪图像。
1. 单击 **[!UICONTROL Save]** 以将图像添加到投票。
1. 将鼠标悬停在用于问题或选择的图像上，然后单击显示的图标以裁剪、删除或替换图像。
