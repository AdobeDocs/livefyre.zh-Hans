# 投票{#polls}

让客户实时发表意见。

投票允许您创建快速、单问投票并将其发布到您的网站。 使用“投票”来鼓励读者参与，收集来自网站访客的反馈、意见和请求。

>[!NOTE]
>
>投票不链接到流。
