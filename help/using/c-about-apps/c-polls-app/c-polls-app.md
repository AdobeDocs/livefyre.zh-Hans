# 投票{#polls}

允许客户实时表达观点。

投票允许您创建快速的单问题投票并将其发布到站点。使用投票鼓励读者参与、收集来自网站访客的反馈、意见和请求。

>[!NOTE]
>
>投票不链接到流。
