# 映射功能{#map-features}

Map应用程序会优惠您网站的这些功能。



* 内容实时输入地图。
* 来自同一区域的内容会进行群集化，并可在画廊模式中查看。
* 缩放到“地图”以集中查看特定区域的内容。
