# 趋势{#trending}

使用趋势分析功能展示特定网站中最活跃的集合，将网站流量推向最活跃的讨论，或让读者一目了然地评估网站活动。 该应用程序显示从最活跃到最不活跃的顶级集合，条形图显示每个集合的相对受欢迎程度。 趋势分析可帮助读者确定最多对话的发生地，并将他们引导到这些对话中。
