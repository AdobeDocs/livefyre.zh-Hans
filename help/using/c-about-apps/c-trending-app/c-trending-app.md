# 趋势趋势{#trending}

使用趋势展示展示特定站点上最活跃的收藏集、推动网站流量到最活跃的讨论或允许读者一览表一览表。应用程序将显示从最多到最不活动的顶级集合，条形图显示显示每个集合的相对受欢迎程度。趋势趋势可帮助读者确定最对话的地点，并引导他们进入这些对话。
