# 聊天{#chat}

在您的网站上实时聊天。

通过聊天，受众可以围绕实时事件、公告或节目进行实时对话。 内容显示为连续的无线聊天流，以促进页面快速参与和激发活动。

福克斯新闻利用聊天捕捉受众对贝尔格里奥当选的看法和反思。 福克斯新闻知道相关且积极的参与将激增，于是在宣布消息发布后的几分钟内，就启动了一次聊天，以立即进行对话。
