# 聊天{#chat}

在您的站点上启用实时聊天。

聊天允许受众围绕实时活动、公告或展示参与实时对话。内容显示为连续的不间断聊天流，可促进页面上快速参与和减缓活动。

Fox News使用“聊天”来捕获教皇Bergoglio的受众观点和反映。由于知道相关且积极参与，Fox News在公告几分钟内发起了一次聊天，旨在立即进行对话。
