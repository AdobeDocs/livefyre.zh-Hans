# 聊天{#chat}

在您的网站上实时聊天。

通过聊天，受众可以参与有关实时事件、公告或演出的实时对话。 内容会以连续的无线聊天流的形式显示，以便于快速参与和活跃页面上的活动。

福克斯新闻利用聊天捕捉受众对贝尔格里奥当选的看法和反思。 福克斯新闻知道相关且积极的参与将激增，于是在宣布消息的几分钟内就发起了聊天，以立即展开对话。
