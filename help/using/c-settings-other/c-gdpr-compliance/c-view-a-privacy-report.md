# 视图隐私报告{#view-a-privacy-report}

视图Livefyre中的隐私报告，查看Livefyre中为用户记录的信息。

1. 转至 **[!UICONTROL Settings > Privacy]**.
1. 单击列表中的隐私报告以对其进行视图。
