# 查看隐私报告{#view-a-privacy-report}

在Livefyre中查看隐私报告，查看Livefyre中为用户记录的信息。

1. 转至 **[!UICONTROL Settings > Privacy]**.
1. 单击列表中的隐私报告可查看它。
