# 查看隐私报告{#view-a-privacy-report}

查看Livefyre中的隐私权报告，了解在Livefyre中为用户录制的信息。

1. **[!UICONTROL Settings > Privacy]**转至。
1. 单击列表中的隐私权报告以查看该报告。
