# 创建和修改翻译集{#create-and-modify-translation-sets}

1. 导览至Studio中的设置。
1. 选择“网络设置”。
1. 选择翻译集。
1. 单击+Set可创建新的翻译集。
1. 为您的集合指定名称和主要语言。
1. 修改要在应用程序名称下显示的文本。 例如，要更改“注释”应用程序使用的字符串的文本，请修改“注释”下的字段。
1. 保存翻译集。
