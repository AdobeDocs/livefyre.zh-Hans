# 将转换集应用于网络{#apply-a-translation-set-to-a-network}

1. 导览至Studio中的设置。 确保您在网络级别工作，而不是站点级别。
1. 选择“网络设置”。
1. 选择“应用程序设置”。
1. 导航到“默认设置”>“翻译集”。
1. 从下拉菜单中选择一个翻译集。
