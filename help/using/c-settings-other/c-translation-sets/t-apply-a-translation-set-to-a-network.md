# 将翻译设置为网络{#apply-a-translation-set-to-a-network}

1. 导航到Studio中的设置。确保您在网络级别工作，而不是在站点级别工作。
1. 选择“网络设置”。
1. 选择“应用程序设置”。
1. 导航到默认设置>翻译集。
1. 从下拉菜单中选择一个翻译集。
