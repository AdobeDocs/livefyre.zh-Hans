# 将翻译集应用于站点{#apply-a-translation-set-to-a-site}

**将翻译集应用到站点**

1. 导览至Studio中的设置。
1. 选择站点。
1. 选择“站点设置”。
1. 选择“应用程序设置”。
1. 导航到“默认设置”>“翻译集”。
1. 从下拉菜单中选择一个翻译集。
