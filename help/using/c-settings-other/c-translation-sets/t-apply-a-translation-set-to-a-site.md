# 将翻译应用于站点{#apply-a-translation-set-to-a-site}

**将翻译应用于站点**

1. 导航到Studio中的设置。
1. 选择站点。
1. 选择站点设置。
1. 选择“应用程序设置”。
1. 导航到默认设置&gt;翻译集。
1. 从下拉菜单中选择一个翻译集。
