# 用户角色和权限{#user-roles-and-permissions}

Livefyre提供多个具有不同权限级别的用户类型。

Studio中提供以下用户角色和列出的访问权限：

>[!NOTE]
>
>对于除管理员以外的所有角色，用户只能访问公共或已与其共享的文件夹或站点。

* **Studio管理员**
   * 访问Studio中的所有权限和权限
   * 管理用户Ress
   * 查看和编辑集成设置
   * 查看和编辑所有公共和专用文件夹、应用程序和流

* **Studio Manager**
   * 创建新站点
   * 邀请用户进入Studio
   * 查看和编辑用户角色
   * 禁止浏览器和IP地址

* **资产经理**
   * 使用社交搜索将资产添加到库
   * 创建和编辑端口到共享文件夹的流
   * 在共享文件夹中创建和编辑共享文件夹和资产

* **用户经理**
   * 查看和编辑最终用户帐户详细信息
   * 停止或白名单最终用户

* **数据管理器：** 授予访问权限以使用Livefyre Analytics仪表板。
* **App Manager**
   * 创建和编辑应用程序
   * 创建和编辑指向应用程序的流
   * 使用社交搜索将资产直接添加到应用程序

* **内容经理：**
   * 使用应用程序内容页面查看和审核内容
   * 添加和编辑标记、标记、备注和权限请求

* **版主：**
   * 访问ModQ
   * 适度调整内容的权利

>[!NOTE]
>
>站点级别用户角色仅向选定站点授予权限。在此级别授予的角色将无法访问其他站点或网络。
