# 登录Studio{#log-into-studio}

网络帐户连接到Studio帐户后，您便可以通过书签链接或任何Livefyre应用程序访问Studio。

1. 从应用程序中，从下拉 **[!UICONTROL Studio]** 列表中 **[!UICONTROL Account]** 选择。
1. Livefyre会将您重定向到Studio登录页面，或者在会话过期时重定向到Studio。
