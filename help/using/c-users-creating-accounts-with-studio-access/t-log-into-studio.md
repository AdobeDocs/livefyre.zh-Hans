# 登录Studio{#log-into-studio}

网络帐户连接到Studio帐户后，您可以通过书签链接或任何Livefyre应用程序访问Studio。

1. 从应用程序中，从&#x200B;**[!UICONTROL Account]**&#x200B;下拉菜单中选择&#x200B;**[!UICONTROL Studio]**。
1. Livefyre会将您重定向到Studio登录页或Studio（如果会话已过期）。
