# 登录Studio{#log-into-studio}

网络帐户连接到Studio帐户后，您可以通过书签链接或任何Livefyre应用程序访问Studio。

1. 从应用程序中，选择 **[!UICONTROL Studio]****[!UICONTROL Account]** 下拉列表。
1. 如果会话过期，Livefyre将重定向到Studio登录页面或Studio。
