# 按URL搜索{#search-by-url}

按URL搜索内容以保存在资产库中。

从Web上的任何URL添加内容。只需输入要嵌入到页面上的URL，然后单击返回。

返回的信息将显示页面的信息卡摘要，包括完整URL、页面标题、图像、署名和第一个段落中的片段。

>[!NOTE]
>
>对于使用URL字段的Instagram搜索，作者将返回为“Instagram”。
