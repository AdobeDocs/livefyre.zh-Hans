# 资产{#assets}

收集、组织和维护选定内容，以便在Livefyre应用程序中使用。

资产是已保存到库中的内容。资产可搜索且可重用，并且可能包括添加的信息，包括标记和重复使用权利。

库允许您将选定内容组织和管理为资产，以便在所有Livefyre应用程序中使用。

库还允许您请求使用收集的社交内容，添加标记以扩充随资产存储的信息，以及组织文件夹中的这些项目，准备好发布到应用程序。

库提供了所有资产的文件夹结构视图，允许您搜索和组织资产。
