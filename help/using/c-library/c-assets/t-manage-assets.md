# 查看资产详细信息{#view-asset-details}

库允许您向保存的内容添加信息，以便根据您的品牌战略组织资产。

1. 将鼠标悬停在单个内容上，并使用窗格中显示的按钮，或 **[!UICONTROL View Details]** 管理单个项目。
1. 选择一条或多条内容
1. 使用页面顶部的按钮管理内容。
