# 2018年月12日{#april}

2018年月12日发行说明发行说明。

## 新增功能 {#section_syx_mdt_wcb}

此版本的生产版本中新增了以下功能：

您现在可以在流中选择智能标签的置信阈值。为标签设置精度得分(0-100)可让您控制检索到的资产的准确性。
