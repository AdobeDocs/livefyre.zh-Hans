# 发行说明- 2018年12月18日

**2018年12月18日更新的发行说明。**

## 增强功能

* 已实施默认情况下启用Instagram商业帐户设置的设置。

## 错误修复

* 修复了使用Instagram规则进行社交搜索时不总是有效的问题。
* 修复了在返回结果时URL搜索看起来不一致的问题。
* 修复了带有韩文字符的流不导入内容的问题。
* 修复了使用Instagram规则的流无法获取新帖子的问题。
