# 2017年9月14日{#september}

2017年9月14日版本的发行说明。

## 生产版本

| **问题类型** | **组件** | **发行说明** |
|---|---|---|
| 错误 | Studio | 修复了阻止权限状态更新在不刷新页面的情况下立即显示的错误。 |
| 增强功能 | Studio | 对图书馆和社交搜索页面的权限模式进行了全面改革。 添加了手动授予和撤销资产权限的功能。 错误／警告消息更加清晰，有助于指导您发送有效的权限请求和回复。 改进的活动历史记录现在显示在模态中。 |
