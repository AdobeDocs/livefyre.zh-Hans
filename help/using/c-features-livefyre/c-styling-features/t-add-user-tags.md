# 向帐户添加用户标记{#add-user-tags-to-an-account}

向帐户添加用户标记以应用用户标记。

1. 通过Studio创建所有者和版主，将&#x200B;**[!UICONTROL Moderator]**&#x200B;用户标记分配给帐户。
1. 创建用户组，然后通过Studio将用户添加到这些用户组，以便将组名称带有的标记应用到选定用户。
1. 使用添加用户标记HTTP调用或Ping for Pull将用户标记直接应用于帐户。
