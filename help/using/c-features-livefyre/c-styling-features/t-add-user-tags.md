# 向帐户添加用户标记{#add-user-tags-to-an-account}

向帐户添加用户标记以应用用户标记。

1. 通过Studio创建所有者和审核者，为帐 **[!UICONTROL Moderator]** 户分配用户标记。
1. 创建用户组，然后通过Studio将用户添加到用户组，以将具有用户组名称的标记应用于选定用户。
1. 使用“添加用户标记HTTP”调用或“Ping for Pull”将用户标记直接应用到帐户。
