# 将用户标记添加到帐户{#add-user-tags-to-an-account}

向帐户添加用户标记以应用用户标记。

1. 通过Studio创建所有者和审核者，将&#x200B;**[!UICONTROL Moderator]**&#x200B;用户标记分配给帐户。
1. 创建用户组，然后通过Studio向用户添加用户，以将具有用户组名称的“标记”应用于选定用户。
1. 使用添加用户标记HTTP调用或Ping for Pull将用户标记直接应用于帐户。
