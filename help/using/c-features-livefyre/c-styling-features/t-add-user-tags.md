# 向帐户添加用户标记{#add-user-tags-to-an-account}

向帐户添加用户标记以应用用户标记。

1. 通过Studio创建所有者和版主，以将 **[!UICONTROL Moderator]** 用户标记分配给帐户。
1. 创建用户组，然后通过Studio向其添加用户，以将标记与选定用户应用组名称。
1. 使用添加用户标记HTTP调用或为拉动的ping，直接将用户标记应用于帐户。
