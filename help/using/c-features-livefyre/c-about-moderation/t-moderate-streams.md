# 限制Twitter用户的内容{#restrict-content-from-twitter-users}

您可以禁止Twitter用户被拖入您的App。**[!UICONTROL Network Settings > Streams]** 此选项卡允许您从流中禁止用户，在提供的字段中输入其Twitter用户名并单击 **[!UICONTROL Add]**。

如果需要，使用该复选框清除列出的用户现有内容。

>[!NOTE]
>
>删除现有内容无法撤消。
