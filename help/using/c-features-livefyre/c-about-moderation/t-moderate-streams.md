# 限制来自Twitter用户的内容{#restrict-content-from-twitter-users}

您可以禁止Twitter用户被拉入您的应用程序。 该选 **[!UICONTROL Network Settings > Streams]** 项卡允许您禁止用户访问您的流，在提供的字段中输入其Twitter用户名并单击 **[!UICONTROL Add]**。

如果需要，使用此复选框可废弃列出用户的所有现有内容。

>[!NOTE]
>
>销毁现有内容的操作无法撤消。
