# 按集合或内容标记进行筛选搜索{#filter-search-by-collection-or-content-tag}

在Studio的“按集合”标签中过滤内容。

1. 在 **[!UICONTROL Tag]****[!UICONTROL Collections]** 面板的搜索下拉列表中选择。
1. 输入用于搜索的标记或单击收藏集下 **[!UICONTROL Tag]** 列出的标记。

   包含选定标记的所有收藏集将显示在列表中。
