# 按“集合”或“内容”标签筛选搜索{#filter-search-by-collection-or-content-tag}

按“集合”标签在Studio中筛选内容搜索。

1. 在面 **[!UICONTROL Tag]** 板的搜索下拉框中 **[!UICONTROL Collections]** 选择。
1. 输入搜索的标记，或单击“集合” **[!UICONTROL Tag]** 下列出的标记。

   包含选定标记的所有集合将显示在列表中。
