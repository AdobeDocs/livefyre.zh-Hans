# 按集合或内容标记筛选搜索{#filter-search-by-collection-or-content-tag}

在Studio中按“集合”标签过滤内容搜索。

1. 在&#x200B;**[!UICONTROL Collections]**&#x200B;面板的搜索下拉列表中选择&#x200B;**[!UICONTROL Tag]**。
1. 输入搜索的标记，或单击集合下列出的&#x200B;**[!UICONTROL Tag]**。

   具有选定标记的所有集合都将显示在列表中。
