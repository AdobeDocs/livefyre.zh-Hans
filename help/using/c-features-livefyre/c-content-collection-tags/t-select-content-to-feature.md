# 从应用程序中选择要添加功能的内容{#select-content-to-feature-from-an-app}

直接从应用程序中功能或取消功能内容。

1. 从应用程序中，将鼠标悬停在要显示的内容上。
1. 单 **[!UICONTROL Feature]** 击以展示内容，或 **[!UICONTROL Unfeature]** 停止展示该内容。
