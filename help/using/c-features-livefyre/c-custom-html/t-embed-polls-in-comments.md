# 在评论中嵌入投票{#embed-polls-in-comments}

在评论或实时博客流中嵌入投票。

1. 在Studio的“App Manager”（应用程序管理器）页 **[!UICONTROL </>]** 面中，单击（嵌入）按钮以生成投票的嵌入代码。
1. 将嵌入代码复制并粘贴到应用 **[!UICONTROL Add Custom HTML]** 程序的字段中。
