# 在评论中嵌入投票{#embed-polls-in-comments}

在评论或Live Blog流中嵌入投票。

1. 在Studio的App Manager页面中，单击 **[!UICONTROL </>]** (embed)按钮以生成投票的嵌入代码。
1. 将嵌入代码复制并粘贴到应用程序的 **[!UICONTROL Add Custom HTML]** 字段中。
