# 在评论中嵌入投票{#embed-polls-in-comments}

在评论或实时博客流中嵌入投票。

1. 在Studio的“App Manager”（应用程序管理器）页面中，单击&#x200B;**[!UICONTROL </>]**（嵌入）按钮以生成投票的嵌入代码。
1. 将嵌入代码复制并粘贴到应用程序的&#x200B;**[!UICONTROL Add Custom HTML]**&#x200B;字段中。
