# 在Studio中查看自定义HTML注释{#view-custom-html-comments-in-studio}

查看使用此过程添加的自定义HTML添加的注释。

>[!NOTE]
>
>无法编辑包含HTML代码的注释或使用Studio添加自定义HTML附件。

1. Studio会显示带有评论的自定义HTML的预览缩略图。
1. 单击缩略图以显示完整附件。
