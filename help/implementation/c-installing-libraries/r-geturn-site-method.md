# 切换站点方法{#geturn-site-method}

此方法返回此站点的URN。

## Java示例 {#section_nyl_ycs_rz}

使用情况：

```
site.getUrn();
```

示例输出：

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```

## NodeJS示例 {#section_xkd_gds_rz}

```

```

示例输出：

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```

