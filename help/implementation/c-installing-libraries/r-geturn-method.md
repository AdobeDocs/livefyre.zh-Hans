# 切换网络方法{#geturn-network-method}

此方法返回此网络的URN。

## Java示例 {#section_nyl_ycs_rz}

```
network.getUrn(); 
```

示例输出：

```
"urn:livefyre:network=`example.fyre.co`" 
```

## NodeJS示例 {#section_xkd_gds_rz}

```
network.getUrn(); 
```

示例输出：

```
"urn:livefyre:network=`example.fyre.co`" 
```

## PHP示例 {#section_ghf_gds_rz}

```
$network->getUrn(); 
```

示例输出：

```
"urn:livefyre:network=`example.fyre.co`" 
```

## Python示例 {#section_dwg_gds_rz}

```
network.urn 
```

示例输出：

```
"urn:livefyre:network=`example.fyre.co`" 
```

## 拼音示例 {#section_enh_gds_rz}

```

示例输出：

```
