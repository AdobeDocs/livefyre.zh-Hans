# getNetworkName网络方法{#getnetworkname-network-method}

此方法返回不带fyre.co的网络名称。

## Java示例 {#section_nyl_ycs_rz}

```
network.getNetworkName();
```

示例输出：

```
"example" 
```

## NodeJS示例 {#section_xkd_gds_rz}

```
network.getNetworkName();
```

示例输出：

```
"example" 
```

## PHP示例 {#section_ghf_gds_rz}

```
$network->getNetworkName(); 
```

示例输出：

```
"example" 
```

## Python示例 {#section_dwg_gds_rz}

```
network.network_name; 
```

示例输出：

```
"example" 
```

示例输出：

"example"
