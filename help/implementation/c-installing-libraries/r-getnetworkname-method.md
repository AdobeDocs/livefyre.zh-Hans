# getNetworkName网络方法{#getnetworkname-network-method}

此方法返回没有fyre. co的网络的名称。

## Java示例 {#section_nyl_ycs_rz}

```
network.getNetworkName();
```

示例输出：

```
"example" 
```

## NodeJS示例 {#section_xkd_gds_rz}

```
network.getNetworkName();
```

示例输出：

```
"example" 
```

```
$network->getNetworkName(); 
```

示例输出：

```
"example" 
```
