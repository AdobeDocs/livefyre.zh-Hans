# buildLiveRetoken网络方法{#buildlivefyretoken-network-method}

返回经过加密的有效Livefyre令牌，该令牌可用于与来自调用该网络的其他Livefyre API交互。

返回经过加密的有效Livefyre令牌，该令牌可用于与来自调用该网络的其他Livefyre API交互。

默认情况下，此令牌设置为在创建时24小时后过期。
