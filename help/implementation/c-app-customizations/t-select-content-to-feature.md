# 从应用程序中选择内容到功能{#select-content-to-feature-from-an-app}

直接从应用程序中执行功能或取消功能内容。

1. 在应用程序中，将鼠标悬停在要功能的内容上方。
1. 单击 **[!UICONTROL Feature]** 以显示内容 **[!UICONTROL Unfeature]** 或停止显示该内容。
