# 从App{#select-content-to-feature-from-an-app}中选择要进行功能的内容

直接从应用程序中功能或取消功能内容。

1. 从应用程序中，将鼠标悬停在要显示的内容上。
1. 单击&#x200B;**[!UICONTROL Feature]**&#x200B;以显示该内容，或单击&#x200B;**[!UICONTROL Unfeature]**&#x200B;以停止显示该内容。
