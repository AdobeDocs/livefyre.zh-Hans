# 从App{#select-content-to-feature-from-an-app}中选择要实现的功能内容

直接从应用程序中功能或取消功能内容。

1. 从应用程序中，将指针悬停在要显示的内容上。
1. 单击&#x200B;**[!UICONTROL Feature]**&#x200B;以设置内容的特征，或单击&#x200B;**[!UICONTROL Unfeature]**&#x200B;以停止设置该内容的特征。
