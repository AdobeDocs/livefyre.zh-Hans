# Sidesk书签{#sidenotes-bookmarklet}

Sides表示可以加载到（几乎）任何现有页面。

要演示，表示使用以下书签：

Sidesk书签

只需将书签复制到书签中，然后单击链接以加载Sidestrap。 系统将提示您输入选择器，该选择器将用于在当前页面上装饰Sidestram。
