# Sidenes Bookmark{#sidenotes-bookmarklet}

可以将Sidones加载到(几乎)任何现有页面上。

要演示Sidones，请使用下面的书签：

Sidenes Bookmark

只需将书签复制到书签中，然后单击链接以加载SideNote。系统将提示您输入用于装饰当前页面上的SiteNote的选择器。
